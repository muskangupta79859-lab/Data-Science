# Data Cleaning Basics

Data cleaning is the process of fixing or removing incorrect, incomplete, or duplicate data to make it ready for analysis.

## Handling Missing Values
- Identifying null or empty values
- Filling missing data with mean, median, or mode
- Removing rows or columns if needed

## Removing Duplicates
- Detecting duplicate rows
- Keeping only unique and correct records

## Why Data Cleaning is Important
Clean data improves accuracy and reliability of analysis results.
