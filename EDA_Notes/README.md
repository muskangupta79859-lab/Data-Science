# Exploratory Data Analysis (EDA) Notes

## What is EDA?
Exploratory Data Analysis is the process of analyzing datasets
to summarize their main characteristics before applying models.

## Why EDA is Important
- Helps understand data distribution
- Identifies missing values and outliers
- Reveals patterns and relationships

## Common EDA Techniques
- Descriptive statistics
- Visualizations (histograms, boxplots)
- Correlation analysis

## Tools Used
- Pandas
- Matplotlib
- Seaborn
